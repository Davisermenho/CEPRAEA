# Adversarial Review Checklist

- [ ] Existe diff real e no escopo.
- [ ] Não houve enfraquecimento/remoção de teste sem justificativa técnica.
- [ ] Comandos declarados possuem resultado verificável.
- [ ] Arquivos fora do escopo não foram alterados sem justificativa.
- [ ] Preview/smoke foi validado quando obrigatório.
- [ ] Logs de execução foram atualizados.
